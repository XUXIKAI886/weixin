使用说明

1

打开微信并登录

确保电脑上已安装微信桌面版并保持登录状态

2

启动微信群发助手

双击运行"微信群发助手.exe"或从源码运行

3

输入登录密码

默认密码：csch9002（建议首次使用后修改）

4

设置消息内容

在左侧面板中输入消息内容、选择接收人和发送选项

5

开始发送

点击"发送消息"按钮，程序将自动完成批量发送

安装使用

方法一：下载可执行文件

  1. 下载发布页面提供的可执行文件
  2. 双击运行"微信群发助手.exe"即可使用
下载最新版本

方法二：从源码运行

  1. 克隆本仓库
    2. git clone https://github.com/XUXIKAI886/weixinqunfazhushou.git
  2. 安装依赖
    3. pip install -r requirements.txt
  3. 运行程序
    4. python main.py

打包成可执行文件

1. 运行打包脚本

python build.py

或者直接双击"打包成可执行文件.bat"

2. 打包完成后，可执行文件将位于"dist"文件夹中

系统要求

  • Windows 7/8/10/11
  • 已安装微信桌面版
  • 如从源码运行：Python 3.7或更高版本
